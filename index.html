Lost in Translation

!! disclaimer !!

Lost in: Translation is a simple choice-based point and click browser game created to raise awareness that machine translations could be detrimental, especially when such translations are used on medical terminologies or how certain words/phrases could be translated in a way where it could be harmful.

As the game progresses, the English used would be progressively changed, simulating the effect as if it has been translated from another language.
